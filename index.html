Usage

Open _config.yml file and start the change:

title: "Milkshake"
description: "Simple, minimal and responsive..."

base_link: '/'
links:
 - name: About
   link: /about
 - name: Blog
   link: /blog

configuration:
 shows:
  usage: 1

Change just what you want. Example if you want to changing title on the above;
you just change the ' title ' object.

(Detail help is exist in the _config.yml file)
